Projects

Arduino Quadcopter

Followed Joop Brokking tutorial and created my first flying project using an Arduino, a gyroscope, a receiver, a transmitter, four ESCs, four motors, a 3S Li-Po battery. The first project where I used soldering.

Smart Thermostat

ESP8266 thermostat for controlling temperature through internet, using a real-time database from Firebase.

Hexacopter

Uses a Pixhawk 2.4.8 flight controller, based on a STM32. By integrating an automatically generated pre-defined flight mission input, this prototype successfully demonstrates autonomous flight capabilities.

PLC Reader with DB

A Windows Forms application designed to establish a connection with a PLC, enabling the user to select specific memory locations, and automatically store the extracted data in a local database at user-defined time intervals. Additionally, the application enables the generation of data outputs within specified time ranges in PDF or CSV formats for enhanced reporting and analysis.

ITEC 2024 Maze Solver and Mapper

Our team designed and built a robot equipped with ultrasonic sensors and a STM32, utilizing Bluetooth for communication. The project involved programming the robot to navigate and map complex mazes effectively. Our practical and efficient approach led to our team winning first place in the competition.

6-DOF Robotic Arm

Built a 6-DOF robotic arm from a DIY kit won at ITEC 2024. The project focused on assembling the mechanical parts, wiring servo motors, and programming basic movements using an Arduino. While not overly complex, it provided hands-on experience with robotic arm mechanics and basic control principles.
